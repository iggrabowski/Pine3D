# Pine3D

###### Lightweight PBR model \& material preview tool



Features

\- ImGui UI.

\- Loading 3D models with using assimp.

\- Toggling texture inputs (normal / roughness / metalness).

\- Sliders for roughness and metalness.

\- Lighting presets.

\- Simplified node-based material preview.







The folder structure should look as follows:

MainFolder/

├─ Pine3D/

│  └─ Pine3D.exe

└─ res/

&nbsp;  └─ shaders/

&nbsp;     ├─ PBR.vert

&nbsp;     └─ PBR.frag



Use "--recursive"" flag when cloning the project to get the submodules as well.
